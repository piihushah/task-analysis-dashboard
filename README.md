# Task Analytics Dashboard

The "Clean-Code" Dashboard (Focus: Fundamentals & Performance) Instead of a standard "To-Do" list, build a Task Analytics Dashboard.

The goal here isn't just the features, but showing how you built it (e.g., using React.memo, useMemo, and custom hooks).

The Concept: A dashboard that tracks productivity tasks and visualizes them. Tech Stack: React, Tailwind CSS, and Recharts (for data visualization).

The "Fake" API: Use JSONPlaceholder or Mockaroo to generate a list of "Completed Tasks" with timestamps
